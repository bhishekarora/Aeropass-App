# Aeropass-App
Hybrid app for Aeropass

![](aeropassapp.png)

## Features
- OTP/Aadhar validation
- Confirm details
- Liveness check
- Save Selfie post liveness check
- Save token
- Upload Boarding pass
- Share with Airport (DCS validation)
- Gate QR code generation
